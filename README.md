# gamehouse
## 概要
React(TypeScript)でブラウザ上で他のユーザーとボードゲームが出来るように開発してみました。(身内用)<br>WebSocketを用いて、常に同期しながらゲームが進行するように開発。

## 開発したゲーム
- クーロンタクティクス
- 花火
- エセ芸術家 ニューヨークへ行く
- ザ・ゲーム
